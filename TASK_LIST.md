# PMT 개발 및 개선 작업 리스트

이 문서는 프로젝트의 개선 사항과 버그 수정 현황을 추적하기 위한 마스터 리스트입니다.

---

## ✅ 완료된 작업 (최근 업데이트: 2026-03-09)

### 🚀 성능 및 안정성 최적화
- [x] **[공통]** PJAX 페이지 전환 시 이벤트 리스너 중복 등록 버그 수정 (내보내기 2회 실행 등 해결)
- [x] **[데이터]** 대용량 데이터(세션/로그) 보존을 위해 **IndexedDB (AppDB)** 기술 적용
- [x] **[백엔드]** 자원사용률 보존 정책(Retention) 비동기 처리로 응답성 향상
- [x] **[백엔드]** 세션 브라우저 데이터 캐싱 적용으로 파일 파싱 부하 감소
- [x] **[수집]** 백그라운드 수집 작업 중첩 방지 로직 강화

### 🎨 UI/UX 및 사용성 개선
- [x] **[공통]** 그리드 최소 높이 확대 (**82vh**)로 데이터 가시성 극대화
- [x] **[설정]** 설정 페이지 레이아웃 개편 (카드 기반의 멀끔한 디자인 적용)
- [x] **[디자인]** 버튼 가독성 개선 (Primary 버튼 흰색 텍스트, 보조 버튼 Subtle Blue 테마)
- [x] **[그리드]** 모든 페이지의 검색/필터 영역 우측 정렬로 레이아웃 통일
- [x] **[실시간]** 히트맵 인터페이스 **IN/OUT 컬럼 분리**로 가독성 향상
- [x] **[실시간]** 현재 수집값 전체 **클립보드 복사** 기능 추가 (탭 구분 형식)
- [x] **[로그조회]** 그리드 컬럼 너비 확대 및 헤더 짤림 현상 수정

### ⚙️ 설정 및 관리 기능 고도화
- [x] **[백업]** 시스템 설정 백업/복구 포맷 변경 (**JSON → Excel**) 및 비개발자용 시트 세분화
- [x] **[설정]** 자원 수집 OID **프리셋 기능** 추가 (MWG, Linux 표준)
- [x] **[프록시]** 프록시 **복제(Clone)** 기능 추가
- [x] **[프록시]** 그룹 내 **OID 일괄 동기화** 기능 추가
- [x] **[자원이력]** 엑셀 내보내기 시 인터페이스 상세 정보를 별도 시트로 분리 저장

### 🛠 기능 변경
- [x] **[공통]** 불필요한 FTP 수집 항목을 **HTTP2**로 전면 교체 (DB 마이그레이션 포함)
