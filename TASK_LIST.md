# PMT 개발 및 개선 작업 리스트

이 문서는 프로젝트의 개선 사항과 버그 수정 현황을 추적하기 위한 마스터 리스트입니다.

---

## 🚀 우선순위: 높음 (기능 오류 및 데이터 결함)
- [ ] **[세션브라우저]** 세션 불러오기 오류 원인 파악 및 수정
- [ ] **[로그조회]** 페이지 재진입 시 "표시할 로그가 없습니다" 문구와 이전 로그가 겹치는 현상 수정

## 🛠 우선순위: 중간 (UX 및 기능 개선)
- [ ] **[공통]** 그리드 필터 초기화 버튼 추가 및 가시성 개선
- [ ] **[세션브라우저]** 그리드 컬럼에 '프록시명' 추가 및 텍스트 개행 이슈 해결 (너비 조정)
- [ ] **[로그조회]** 그리드 컬럼명 잘림 이슈 해결
- [ ] **[실시간]** 히트맵 좌우 너비 자동 조절 로직 적용
- [ ] **[공통]** 정보 표시 인디케이터(세션/로그) 설정바로 이동
    - [ ] 세션브라우저 정보 인디케이터 및 내보내기 버튼
    - [ ] 로그조회 정보 인디케이터

## 🎨 우선순위: 낮음 (UI/디자인 및 최적화)
- [ ] **[공통]** 주요 액션 버튼 색상 변경 (흰색 -> 푸른색) (진행 중)
- [ ] **[세션브라우저]** 상세분석/목록 돌아가기 버튼 스타일 통일 (텍스트 링크 형태)
- [ ] **[공통]** 네비바 메뉴 이동 시 깜빡임 및 스타일 소실 현상 개선 (SPA 라이크 방식 검토 또는 캐싱 최적화)
