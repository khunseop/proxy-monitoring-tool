# 릴리즈 노트

## v1.7.0 (2026-03-31)
- **통합 분석 대시보드 및 시각화 강화**
  - **로그 분석 통합**: '로그 원격 조회' 페이지 내에 분석 탭을 통합하여 조회 결과를 즉시 통계 차트로 확인 가능
  - **세션 분석 추가**: 세션 브라우저에 '결과 분석' 탭을 신설하여 클라이언트/서버별 세션 분포 및 트래픽 요약 제공
  - **이력 그래프 분석**: 자원사용률 이력 조회 시 데이터를 지표별/인터페이스별 개별 라인 차트로 시각화하는 기능 추가
- **SNMP 및 성능 개선 (고속 회선 대응)**
  - **Counter64 지원**: HTTP, HTTPS, HTTP2, Blocked 카운터를 64비트(Counter64) OID로 업데이트하여 대량 트래픽 수집 안정화
  - **1Gbps+ 회선 최적화**: 인터페이스 OID를 High Capacity(ifHCInOctets 등)로 변경하여 1Gbps 이상 대역폭에서의 카운터 래핑(Wrap-around) 문제 해결
  - **계산 로직 정교화**: Mbps 산출 시 카운터 리셋 및 래핑 감지 로직을 강화하여 데이터 정확도 향상
- **UI/UX 대규모 개편 (일관성 및 편의성)**
  - **다이나믹 그리드 레이아웃**: 모든 테이블과 히트맵에서 내부 스크롤을 제거하고 데이터 양에 따라 높이가 자동 조절되도록 개선 (브라우저 메인 스크롤 활용)
  - **통일된 페이지 디자인**: 모든 메뉴의 타이틀, 서브 탭, 설정 바 레이아웃을 표준화하여 일관된 사용자 경험 제공
  - **조회 편의 기능**: 프록시 선택 '모두해제' 버튼 추가, 자원이력 조회 시 시간 범위 프리셋(1h, 6h, 1d, 7d) 제공
  - **검색 최적화**: 로그 및 세션 조회의 검색어(grep -F) 입력창 너비를 확장하여 시인성 개선
- **데이터 활용도 개선**
  - **Raw 데이터 내보내기**: CSV 내보내기 시 Bytes, Seconds 등의 숫자 필드를 원본 값으로 보존하여 엑셀 정렬 및 피벗 분석 용이성 확보
- **버그 수정 및 리팩토링**
  - **코드 모듈화**: 거대해진 `resource_usage.py`를 서비스 레이어로 분리하여 유지보수성 향상
  - **즉시 업데이트**: 조회 후 새로고침 없이 그리드가 바로 갱신되도록 수집 로직 및 가시성 버그 수정
  - **안정성**: 세션 상세보기 실패, DB 스키마 불일치(500 에러) 등 보고된 주요 버그 해결

## v1.6.0 (2026-03-09)
- **성능 및 안정성 최적화**
  - 대용량 데이터(세션/로그) 보존을 위해 브라우저 **IndexedDB (AppDB)** 기술 적용 (페이지 전환 후에도 수만 건의 데이터 유지 가능)
  - PJAX 페이지 전환 시 이벤트 리스너 중복 등록 및 내보내기 중복 실행 버그 수정
  - 백엔드 자원사용률 보존 정책(Retention)을 비동기 처리로 전환하여 수집 성능 향상
  - 세션 브라우저 데이터 캐싱 도입으로 실시간 조회 시 파일 파싱 부하 대폭 감소
- **UI/UX 및 디자인 고도화**
  - 모든 표 그리드 높이를 **82vh**로 확대하여 데이터 가시성 확보
  - 설정(Settings) 페이지 레이아웃을 카드 기반의 현대적인 디자인으로 전면 개편
  - 버튼 가독성 개선: 진한 배경 버튼에 흰색 텍스트 적용, 보조 버튼에 Subtle Blue 테마 적용
  - 모든 페이지의 검색/필터 영역을 우측 정렬하여 레이아웃 통일성 확보
  - 실시간 히트맵의 인터페이스 지표를 **IN / OUT 컬럼으로 분리**하여 가독성 향상
  - 실시간 수집값 전체를 탭 구분 형식으로 **클립보드 복사**하는 기능 추가
- **설정 관리 기능 고도화**
  - 시스템 백업/복구 포맷을 **JSON에서 엑셀(.xlsx)로 변경** 및 비개발자용 시트 세분화
  - 자원 수집 OID **프리셋 기능** 추가 (MWG 표준, Linux 표준 지원)
  - 프록시 장비 **복제(Clone)** 및 그룹 내 **OID 일괄 동기화** 기능 추가
  - 자원 이력 엑셀 내보내기 시 인터페이스 상세 정보를 별도 시트로 분리하여 저장
- **기능 및 환경 업데이트**
  - 자원 수집 항목 중 FTP를 **HTTP2**로 전면 교체 (DB 마이그레이션 포함)
  - 앱 버전 정보 업데이트: 2026.03.09
