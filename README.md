# PMT (프록시 성능 분석 도구) 사용자 매뉴얼

PMT는 여러 프록시 서버의 상태와 성능을 한눈에 파악하고, 장애 원인을 신속하게 분석할 수 있도록 돕는 웹 기반 관리 도구입니다. 복잡한 명령어 없이 웹 UI를 통해 직관적으로 프록시를 운영하고 분석할 수 있습니다.

## 주요 기능 소개

PMT는 다음과 같은 핵심 기능을 제공합니다.

-   **⚙️ 설정 (프록시/그룹 관리)**: 모니터링할 프록시 서버와 그룹을 등록하고 시스템 전역 설정을 관리합니다.
-   **📊 자원 모니터링**: 
    - **실시간 모니터링**: 프록시들의 CPU, 메모리, 회선 사용량 등을 실시간 히트맵과 그래프로 확인합니다.
    - **자원 이력 및 그래프 분석**: 과거의 자원 사용 기록을 조회하고, 지표별/회선별 시계열 차트로 상세 트렌드를 분석합니다.
-   **👤 세션 브라우저**: 프록시 서버의 현재 활성 세션(접속자) 정보를 조회하고, 클라이언트/서버별 세션 분포를 시각적으로 분석합니다.
-   **📈 로그 분석 및 조회**: 프록시 원격 로그를 직접 조회하거나 파일을 업로드하여 과부하 원인, 주요 접속자, 인기 사이트 등을 분석합니다.

## 사용 방법

### 1. ⚙️ 프록시 설정 및 관리

분석할 프록시 서버 정보를 등록하고 수집 기준을 설정합니다.

1.  **그룹 및 프록시 등록**: 장비를 그룹별로 나누어 관리할 수 있으며, SSH 및 SNMP 정보를 입력하여 등록합니다.
2.  **OID 프리셋 설정**: `설정` 메뉴에서 MWG 및 Linux 표준 프리셋을 제공하여 클릭 한 번으로 수집 지표(Counter64 대응)를 자동 설정할 수 있습니다.
3.  **UI 설정**: 그리드에 표시할 기본 행 수(20~200행)를 사용자의 해상도에 맞춰 조절할 수 있습니다.
4.  **백업 및 복구**: 모든 시스템 설정을 **Excel(.xlsx)** 파일로 안전하게 백업하거나 복구할 수 있습니다.

### 2. 📊 자원 모니터링 및 이력 분석

프록시의 성능 상태를 다각도로 모니터링합니다.

-   **실시간 모니터링**: 히트맵을 통해 여러 장비의 상태를 색상으로 직관적으로 파악하며, 이상 징후 발생 시 실시간 시계열 그래프로 세부 추이를 확인합니다.
-   **이력 그래프 분석**: 특정 기간의 자원 이력을 조회하고, 이를 **지표별/회선별 독립 그래프**로 변환하여 분석할 수 있습니다. 1Gbps 이상의 고속 회선 트래픽도 정확하게 측정됩니다.
-   **자동 레이아웃**: 데이터 양에 따라 그리드와 히트맵 높이가 자동으로 조절되어 별도의 내부 스크롤 없이 페이지 전체를 넓게 활용할 수 있습니다.

### 3. 👤 세션 브라우저 및 통계 분석

"지금 이 순간 누가 프록시를 사용하고 있는지"와 통계적 분포를 확인합니다.

1.  **세션 조회**: 여러 프록시의 활성 세션을 통합 조회하고, 컬럼별 필터링을 통해 특정 사용자나 사이트를 찾습니다.
2.  **결과 분석**: '결과 분석' 탭에서 클라이언트별 세션 점유율, 프로토콜 분포, 총 전송량 등의 통계를 차트로 확인합니다.
3.  **상세 정보**: 특정 세션을 더블 클릭하여 모든 기술적 세부 정보를 확인할 수 있습니다.

### 4. 📈 로그 분석 및 조회

장애 원인 규명을 위해 원격 로그를 실시간 조회하거나 정밀 분석합니다.

-   **원격 로그 조회 (SSH)**: 서버에 접속하지 않고도 웹에서 직접 로그를 grep 검색하여 수집합니다. (최대 10,000줄 지원)
-   **통합 분석 대시보드**: 조회된 로그 결과를 기반으로 **'분석하기'** 버튼을 눌러 Top 클라이언트, Top 호스트, 차단 사유 분포 등을 즉시 시각화합니다.
-   **데이터 내보내기**: 분석 결과 및 조회된 로그를 CSV 파일로 내보낼 수 있으며, Bytes 등 숫자 정보는 원본 값으로 보존되어 엑셀에서 즉시 정렬/통계가 가능합니다.
